<h1 align="center">Fork of Jellyfin Kistu Plugin</h1>

## About

This plugin adds the metadata provider for [Kitsu](https://kitsu.app) along with title language selection and additional metadata not passed in the upstream version.

## Installation

Add repository: https://github.com/nepellion/jellyfin-plugin-kitsu/raw/refs/heads/master/static/manifest.json
Install "Kitsu (nep fork)" from the repository.
